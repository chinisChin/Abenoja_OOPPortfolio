Activity #1: Variables

Learnings & Reflections:

In this activity, I investigated how Java handles data using the Stack and Heap. I discovered that the actual object data (such the name "John" or "Maria") is allocated in the Heap, whilst local reference variables like s1, s2, and s3 are stored on the Stack.

Understanding object references was a crucial lesson: when I assigned s3 = s2, it only made s3 point to the same memory place as s2 rather than creating a new object. As a result, changing the name on S3 to "Ana" also altered the name on S2, highlighting the significance of exercising caution when it comes to object aliasing.
I had a hard time understanding the memory locations on this activity but as I look back it is just easy. And I probably would figure out it if I do the activities right now.

Activity #3: Simple Wallet System

package javaUno;

import java.util.Scanner;

public class SimpleWalletAbenoja {

	public static void main(String[] args) {
		
		double initial = 3000.00;
		int count = 1;
		boolean correct = true;
		
		Scanner input = new Scanner(System.in);
		
		while (correct) {
			
			System.out.println("=====WALLET MENU====");
			System.out.println("1. View Balance");
			System.out.println("2. Add Money");
			System.out.println("3. Spend Money");
			System.out.println("4. Exit");
			System.out.println("Please select one (1-4): ");
			
			int choice = input.nextInt();
			
			switch(choice) {
			case 1:
				System.out.println("Transaction "+ count);
				System.out.println("Your current balance is : " + initial);
				count++;
				break;
				
			case 2:
				System.out.println("Transaction "+ count);
				System.out.println("Enter Amount: ");
				double curr = input.nextDouble();
				if(curr >= 0) {
					initial+=curr;
					System.out.println("Added Amount Succesfully!");
					count++;
				}else {
					System.out.println("Invalid Amount!!!");
				}
				break;
				
			case 3:
				System.out.println("Transaction "+ count);
				System.out.println("Enter Amount to Spend: ");
				double spend = input.nextDouble();
				if(spend > initial) {
					System.out.println("Error: Amount exceeds balance");
				}else if (spend > 1000.00) {
					System.out.println("Error:Maximum spending amount is only 1,000");
				}else {
					initial -= spend;
					System.out.println("Money Spent Succesfully!");
					count++;
				}
				break;
			case 4:
				System.out.println("Exiting Wallet System");
				System.out.println("Transaction Count is "+ --count);
				correct = false;
				break;
				
			default:
				System.out.println("Invalid Choice. Please choose 1-4 ONLY!");
		
			}
			
			
		}

	}

}

Learnings & Reflections:

This activity is where we learned to use the user input methods, which are a big help to my programming journey. It was "for sure" something that I will use in the near future.

Activity #5: Personal Expense Tracker

    package javaUno;

import java.util.Scanner;

public class ExpenseTrackerAbenoja {
	
	public static void display() {
		System.out.println("====Expense Tracker====");
	}
	
    public static int Total(int food, int transpo, int other) {
    	
    	int sum = food + transpo + other;
    	
    	return sum;
    }
    
    public static String budget(int expense) {
    	
    	int budget = 5000;
    	
    	if(budget < expense) {
    		return "Exceeded your budget";
    	}
    	return "Within the budget";
    }
    
    public static void both(int Totexpense, String status) {
    	System.out.println("Your total expenses for the food, transportation and others is " + Totexpense);
    	
    	
    	System.out.println("Your Budget Status is: " + status);
    }
    
    public static int remain(int total) {
    	
    	int current = 5000 - total;
    	
    	return current;
    }
	public static void main(String[] args) {
		
		Scanner input = new Scanner(System.in);
		
		display();
		
		System.out.println("List down your expenses: ");
		System.out.println("Food: ");
		int food = input.nextInt();
		
		System.out.println("Transportation: ");
		int transpo = input.nextInt();
		
		System.out.println("Other Expenses: ");
		int other = input.nextInt();
		
		int result = Total(food, transpo, other);
		
		System.out.println("The total expense is: " + result);
		
		String status = budget(result);
		System.out.println("Did you exceed or not?: " + status);
		
		both(result, status);
		
		int current = remain(result);
		System.out.println("Your remaining budget is : "+ current);
		
	}

}

Learnings & Reflections:

I honestly just enjoyed the process on creating this activity, it just required me a lot of methods but all in all it was just an easy coding activity for me after going through the past activities.

Seatwork #2: Basic ATM System

    package javaUno;

import java.util.Scanner;

public class ATMTransactionCounterAbenoja {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
		double balance = 8000.0;
        int count = 0;
        boolean active = true;

        Scanner input = new Scanner(System.in);
      
        while (active) {
            // 1. Display Menu
            System.out.println("\n===== ATM MENU =====");
            System.out.println("1. Check Balance");
            System.out.println("2. Deposit");
            System.out.println("3. Withdraw");
            System.out.println("4. Exit");
            System.out.print("Select: ");

    
            int input = input.nextInt();
            
     
            switch (input) {
                case "1":
                    System.out.println("Current Balance: PHP " + balance);
                    count++;
                    break;

                case "2":
                    System.out.print("Enter deposit amount: ");
                    // Parsing the string input to a double
                    double dep = Double.parseDouble(System.console().readLine());
                    if (dep > 0) {
                        balance += dep;
                        System.out.println("Deposit successful!");
                        count++;
                    } else {
                        System.out.println("Invalid deposit amount.");
                    }
                    break;

                case "3":
                    System.out.print("Enter withdrawal amount: ");
                    double draw = Double.parseDouble(System.console().readLine());

                    // 3. if-else for validating withdrawal constraints
                    if (draw > 2000) {
                        System.out.println("Error: Maximum withdrawal per transaction is 2,000 pesos.");
                    } else if (draw > balance) {
                        System.out.println("Error: Amount exceeds balance.");
                    } else if (draw <= 0) {
                        System.out.println("Error: Invalid amount.");
                    } else {
                        balance -= draw;
                        System.out.println("Withdrawal successful.");
                        count++;
                    }
                    break;

                case "4":
                    // 4. Display total transactions performed before exiting
                    System.out.println("Exiting program...");
                    System.out.println("Total transactions performed: " + count);
                    active = false;
                    break;

                default:
                    System.out.println("Invalid option. Please choose 1-4.");
            }
        }
    }
}
